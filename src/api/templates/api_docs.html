API Documentation

Integrate AI-powered breast cancer detection into your applications

Overview

The AI Health Assistant provides a REST API for breast cancer risk assessment using machine learning. The API accepts medical measurements and returns risk predictions with confidence scores.

ML-Powered

Advanced Random Forest model trained on medical datasets

Secure

RESTful API with proper error handling and validation

Fast

Real-time predictions with sub-second response times

Accurate

High accuracy model with confidence scoring

Endpoints
POST /predict

Make a breast cancer risk prediction

Request Body:

{ "features": [ 12.32, 12.39, 78.85, 464.1, 0.10280, 0.06981, 0.03987, 0.03700, 0.1959, 0.05955, 0.2360, 0.6656, 1.670, 17.43, 0.008045, 0.011800, 0.01683, 0.012410, 0.01924, 0.002248, 13.50, 15.64, 86.97, 549.1, 0.1385, 0.1266, 0.12420, 0.09391, 0.2827, 0.06771 ] }

Response:

{ "prediction": 0, "probability": 0.123, "status": "success" }

Prediction: 0 = Low Risk (Benign), 1 = High Risk (Malignant)

Probability: Confidence score between 0.0 and 1.0

GET /health

Check API health status

Response:

{ "status": "healthy", "model_loaded": true }
Feature Order

The features array must be in the exact order shown below (30 features total):

1. mean radius 11. mean concave points 2. mean texture 12. radius error 3. mean perimeter 13. texture error 4. mean area 14. perimeter error 5. mean smoothness 15. area error 6. mean compactness 16. smoothness error 7. mean concavity 17. compactness error 8. mean concave points 18. concavity error 9. mean symmetry 19. concave points error 10. mean fractal dimension 20. symmetry error 21. fractal dimension error 22. worst radius 23. worst texture 24. worst perimeter 25. worst area 26. worst smoothness 27. worst compactness 28. worst concavity 29. worst concave points 30. worst symmetry 31. worst fractal dimension
Example Usage

Python Example:

import requests url = "http://localhost:5000/predict" data = { "features": [ 12.32, 12.39, 78.85, 464.1, 0.10280, 0.06981, 0.03987, 0.03700, 0.1959, 0.05955, 0.2360, 0.6656, 1.670, 17.43, 0.008045, 0.011800, 0.01683, 0.012410, 0.01924, 0.002248, 13.50, 15.64, 86.97, 549.1, 0.1385, 0.1266, 0.12420, 0.09391, 0.2827, 0.06771 ] } response = requests.post(url, json=data) result = response.json() print(f"Prediction: {result['prediction']}") print(f"Probability: {result['probability']:.3f}")

JavaScript Example:

fetch('/predict', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ features: [ 12.32, 12.39, 78.85, 464.1, 0.10280, 0.06981, 0.03987, 0.03700, 0.1959, 0.05955, 0.2360, 0.6656, 1.670, 17.43, 0.008045, 0.011800, 0.01683, 0.012410, 0.01924, 0.002248, 13.50, 15.64, 86.97, 549.1, 0.1385, 0.1266, 0.12420, 0.09391, 0.2827, 0.06771 ] }) }) .then(response => response.json()) .then(data => { console.log('Prediction:', data.prediction); console.log('Probability:', data.probability); });
